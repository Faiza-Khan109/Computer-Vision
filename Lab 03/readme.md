**🖼️ OpenCV Computer Vision Lab**



A collection of Computer Vision tasks implemented using Python \& OpenCV in Jupyter Notebook.



**📌 Features**



✔ Image loading and transformation



✔ Grayscale conversion and resizing



✔ Automatic brightness \& contrast enhancement



✔ Image similarity detection (MSE \& SSIM)



✔ HSV-based color detection



✔ License plate blurring for privacy



✔ Custom watermarking (centered, scalable, opacity control)



✔ OCR preprocessing (noise removal + thresholding)



✔ 2×2 collage generator



✔ Batch grayscale converter



✔ Noise reduction in scanned documents



**🔄 Processing Flow**



All modules follow a consistent pattern:



Load → Preprocess → Apply Operation → Display (Matplotlib) → Save Output



**🛠️ Libraries**



OpenCV



NumPy



Matplotlib



Scikit-Image

Demonstrates practical image processing techniques for:



Enhancement

\# Edge Detection in Computer Vision



This project shows how computers detect \*\*edges in images\*\* using Python and OpenCV.



Edge detection is important in many AI systems like:

\- Self-driving cars

\- Medical imaging

\- Face recognition

\- Document analysis



\## Edge Detection Methods Used



\### Sobel Edge Detection

Detects edges by measuring how fast the brightness of the image changes.



\### Prewitt Edge Detection

Another method for finding edges using filters.



\### Canny Edge Detection

A powerful edge detection method widely used in real-world applications.



\## Real World Applications in this Project



Medical Images

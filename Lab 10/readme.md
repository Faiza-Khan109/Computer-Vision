\# Lab Assignment: Object Detection Using YOLOv8



\## Objective

To implement object detection on real-world case studies using YOLOv8 and analyze results using image-based processing.



\## Tools \& Technologies

\- YOLOv8 (Ultralytics)

\- OpenCV

\- Python

\- Matplotlib



\## Case Studies Implemented



1\. Smart Classroom Monitoring

2\. Smart Parking Surveillance System

3\. Wildlife Conservation Monitoring

4\. Industrial Defect Detection System

5\. Retail Shelf Monitoring System



\## Methodology

\- Load YOLOv8 pretrained model

\- Read input images

\- Perform object detection

\- Visualize results with bounding boxes



\## Results

All case studies successfully detect objects and display annotated images.



\## Conclusion

The experiment demonstrates effective use of YOLOv8 for real-world image-based object detection applications.
